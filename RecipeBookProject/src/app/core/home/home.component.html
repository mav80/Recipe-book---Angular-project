Welcome to the Recipe book!

This simple app allows you to create your own recipes. Each recipe consists of a name, description, link to an image and a list of ingredients. There is also a shopping list in which you can put ingredients manually, or directly from a recipe details component.

Upon registering and signing in user can also store recipes in online database and fetch them from there by using Manage recipes menu in the top right corner.
